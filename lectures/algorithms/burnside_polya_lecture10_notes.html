1. Зачем нужна лемма Бёрнсайда

Во многих задачах нужно посчитать количество объектов с точностью до симметрии. Например, две раскраски ожерелья могут отличаться только поворотом. Если физически повернуть ожерелье, оно остаётся тем же самым ожерельем. Значит, обычный подсчёт всех массивов длины n переучитывает один и тот же объект много раз.

Пример. Есть бинарная строка длины 4: 1000. При циклических сдвигах получаем: 
1000 → 0100 → 0010 → 0001
Если мы считаем строки как массивы, это 4 разных объекта. Если считаем ожерелья с точностью до поворота, это один объект.

Основная идея лекции: симметрии образуют группу G, которая действует на множестве объектов X. Объекты, переходящие друг в друга действиями группы, считаются одинаковыми. Нужно посчитать число классов эквивалентности, то есть число орбит.

Лемма Бёрнсайда:
число орбит = (1 / |G|) · Σg∈G |Fix(g)|,
где Fix(g) — множество объектов, которые не меняются под действием g.

2. Моноид, группа, перестановки

2.1. Моноид

Моноид — это множество M с бинарной операцией ⋅, для которой выполнены два свойства:

  1. Ассоциативность: (a⋅b)⋅c = a⋅(b⋅c).
  2. Нейтральный элемент: существует e, такой что e⋅a = a⋅e = a.

В моноиде не обязаны существовать обратные элементы.

Примеры моноидов:
  • натуральные числа с операцией сложения и нейтральным элементом 0;
  • строки с операцией конкатенации и пустой строкой;
  • функции из множества в себя с операцией композиции и тождественной функцией.

2.2. Группа

Группа — это моноид, где у каждого элемента есть обратный. Формально множество G с операцией ⋅ является группой, если:

  1. Замкнутость: если a,b∈G, то a⋅b∈G.
  2. Ассоциативность: (a⋅b)⋅c = a⋅(b⋅c).
  3. Нейтральный элемент: существует e∈G, такой что e⋅a=a⋅e=a.
  4. Обратный элемент: для каждого a∈G существует a^{-1}∈G, такой что a⋅a^{-1}=a^{-1}⋅a=e.
В комбинаторике симметрий чаще всего группы появляются как группы перестановок. Элемент группы — это действие, которое переупорядочивает позиции объекта. Например, поворот ожерелья, отражение прямоугольника, перестановка элементов массива.

2.3. Перестановка как биекция

Перестановка множества {1,2,...,n} — это биекция этого множества в себя. Иначе говоря, каждому старому индексу она сопоставляет новый индекс, причём никакие два элемента не склеиваются.

Перестановки удобно записывать так: 

π = [2, 3, 1, 4]

Это значит: 

π(1)=2, π(2)=3, π(3)=1, π(4)=4.

Все перестановки на n элементах образуют группу S_n, называемую симметрической группой. Операция — композиция перестановок.

3. Действие группы на множестве

Пусть есть группа G и множество объектов X. Действие группы на множестве — это правило, которое каждому элементу группы g∈G и каждому объекту x∈X сопоставляет новый объект g·x∈X.

Действие должно удовлетворять двум условиям:

  1. Нейтральный элемент ничего не меняет: e·x = x.
  2. Композиция действий согласована с операцией группы: (gh)·x = g·(h·x).
Пример. Пусть X — множество массивов длины n, а G — группа циклических сдвигов. Тогда действие сдвига на массив — это перенос элементов по кругу. Например: 
[a,b,c,d] --сдвиг на 1--> [d,a,b,c]

Важно: группа действует не обязательно на себе. Она может действовать на геометрических фигурах, раскрасках, строках, массивах, графах, точках плоскости и так далее.

4. Перестановки и циклы

Любую перестановку можно разложить на независимые циклы. Цикл показывает, как элементы переходят друг в друга при многократном применении перестановки.

Рассмотрим перестановку: 
π(1)=3, π(3)=5, π(5)=1, π(2)=4, π(4)=2
Тогда циклическая запись: 
π = (1 3 5)(2 4)
Первый цикл длины 3, второй цикл длины 2.

Циклы важны для теоремы Пойа. Если перестановка действует на позиции раскраски, то чтобы раскраска была неподвижна, все позиции внутри одного цикла должны иметь одинаковый цвет.

Если перестановка имеет c(π) циклов, а доступно k цветов, то число раскрасок, неподвижных относительно этой перестановки, равно:
k^{c(π)}.
Доказательство. Пусть позиции внутри одного цикла переходят друг в друга. Если раскраска не меняется после перестановки, то цвет первой позиции цикла должен совпадать с цветом второй, второй с третьей и так далее. Значит, весь цикл обязан иметь один цвет. Для каждого цикла можно независимо выбрать один из k цветов. Если циклов c(π), получаем k·k·...·k = k^{c(π)}.

5. Композиция действий и замкнутость

5.1. Композиция действий

Если у нас есть два действия g и h, их композиция означает: сначала применить h, потом g. В групповой записи это обычно пишут как g∘h или просто gh.

Пусть массив [1,2,3,4]. Сначала сдвиг на 1 вправо: 
[1,2,3,4] → [4,1,2,3]
Потом ещё сдвиг на 1 вправо: 
[4,1,2,3] → [3,4,1,2]
Композиция двух сдвигов на 1 — это сдвиг на 2.

5.2. Замкнутая группа действий

Множество действий замкнуто, если композиция любых двух действий из множества снова лежит в этом множестве. Для группы это обязательное свойство.

Например, все циклические сдвиги массива длины n образуют группу:
  • сдвиг на 0 — нейтральный элемент;
  • сдвиг на a после сдвига на b — это сдвиг на a+b mod n;
  • обратный к сдвигу на a — сдвиг на n-a.

5.3. Пример незамкнутого набора: элементарные транспозиции

Элементарная транспозиция — это swap соседних элементов, например s_i=(i,i+1). Набор только элементарных транспозиций сам по себе не является группой, потому что он не замкнут относительно композиции.

В S_3 возьмём соседние swap: 
s1 = (1 2), s2 = (2 3)
Их композиция: 
s1 ∘ s2
даёт цикл длины 3, например (1 2 3) или (1 3 2) в зависимости от порядка композиции. Это уже не элементарная транспозиция. Значит, множество {s1,s2} не замкнуто.

Однако если взять все композиции элементарных транспозиций, то получится вся группа перестановок S_n. Поэтому говорят, что элементарные транспозиции порождают симметрическую группу, но сами не образуют группу без замыкания.

6. Эквивалентность, орбиты, стабилизаторы

6.1. Отношение эквивалентности с точностью до действия группы

Пусть группа G действует на множестве X. Будем считать два объекта x,y∈X эквивалентными, если существует действие g∈G, переводящее один объект в другой:

x ∼ y ⇔ ∃g∈G: g·x = y.
Докажем, что это отношение эквивалентности.
  1. Рефлексивность: x∼x, потому что нейтральный элемент e даёт e·x=x.
  2. Симметричность: если x∼y, то g·x=y. Так как g обратим, применяем g^{-1}: g^{-1}·y=x. Значит, y∼x.
  3. Транзитивность: если x∼y и y∼z, то g·x=y, h·y=z. Тогда (h g)·x=h·(g·x)=h·y=z. Значит, x∼z.

6.2. Орбита

Орбита элемента x — это множество всех объектов, которые можно получить из x действиями группы:

Orb(x) = { g·x | g∈G }.

Орбита — это класс эквивалентности объекта x. Поэтому число разных объектов с точностью до симметрий — это число орбит.

6.3. Стабилизатор

Стабилизатор элемента x — это множество действий группы, которые оставляют x на месте:

Stab(x) = { g∈G | g·x = x }.

Стабилизатор — это подгруппа группы G.

Доказательство.
  • Нейтральный элемент стабилизирует любой x, потому что e·x=x.
  • Если g·x=x и h·x=x, то (gh)·x=g·(h·x)=g·x=x. Значит, композиция тоже в стабилизаторе.
  • Если g·x=x, то применяя g^{-1}, получаем g^{-1}·x=x. Значит, обратный элемент тоже в стабилизаторе.
Следовательно, Stab(x) — подгруппа.

6.4. Неподвижная точка

Для фиксированного действия g∈G объект x называется неподвижной точкой, если:

g·x = x.

Множество всех неподвижных точек действия g обозначают:

Fix(g) = { x∈X | g·x = x }.

7. Примеры действий групп

7.1. Точки на плоскости и изменение масштаба

Пусть X = R^2, а группа состоит из преобразований масштабирования:

s_λ(x,y) = (λx, λy), где λ≠0.

Композиция масштабирований:

s_λ ∘ s_μ = s_{λμ}.

Нейтральный элемент — λ=1. Обратный — λ^{-1}.

Орбиты: если разрешены все положительные масштабы, то точка движется по лучу из начала координат. Нулевая точка (0,0) остаётся неподвижной при любом масштабировании.

7.2. Точки на плоскости и поворот

Пусть группа — повороты вокруг начала координат на углы, кратные 90°:

G = {0°,90°,180°,270°}.

Это группа из четырёх элементов. Орбита обычной точки состоит из четырёх точек. Но точка (0,0) фиксируется всеми поворотами, поэтому её стабилизатор — вся группа.

7.3. Массивы и перестановки

Пусть X — множество массивов длины n. Перестановка позиций действует на массив, переставляя его элементы. 

array = [a1,a2,a3]
π = (1 2 3)
π·array = [a3,a1,a2]   // один из вариантов соглашения о направлении

Важно: в задачах нужно один раз выбрать соглашение о направлении действия. Для количества орбит это обычно не влияет, потому что группа всё равно содержит обратные действия.

7.4. Массивы и циклические сдвиги

Циклические сдвиги длины n образуют группу C_n. Элемент r^t — сдвиг на t позиций.

Для строки длины 6 сдвиг на 2 позиции разбивает позиции на циклы:

0 → 2 → 4 → 0
1 → 3 → 5 → 1

То есть у перестановки два цикла длины 3. Если цветов k, то неподвижных раскрасок будет k^2: один цвет выбирается для первого цикла, один для второго.

8. Сумма мощностей стабилизаторов и сумма неподвижных точек

Это ключевая техническая лемма, из которой почти сразу следует лемма Бёрнсайда.

Σx∈X |Stab(x)| = Σg∈G |Fix(g)|.

8.1. Идея через таблицу

Рассмотрим таблицу, где строки — элементы x∈X, столбцы — действия g∈G. В ячейке ставим 1, если g·x=x, и 0 иначе.

g₁g₂g₃...
x₁1/01/01/0...
x₂1/01/01/0...
x₃1/01/01/0...

Если суммировать единицы по строке x, мы считаем, сколько действий фиксируют x, то есть |Stab(x)|.

Если суммировать единицы по столбцу g, мы считаем, сколько объектов фиксирует действие g, то есть |Fix(g)|.

Доказательство. Обе суммы считают одно и то же множество пар:
P = { (x,g) ∈ X×G | g·x=x }.
Если группировать пары по x, получаем Σ_x |Stab(x)|. Если группировать пары по g, получаем Σ_g |Fix(g)|. Так как множество пар одно и то же, суммы равны.

9. Лемма Бёрнсайда

9.1. Формулировка через неподвижные точки

Пусть конечная группа G действует на конечном множестве X. Тогда число орбит равно:
|X/G| = (1/|G|) Σg∈G |Fix(g)|.

Здесь X/G — множество орбит, то есть множество объектов с точностью до действия группы.

9.2. Формулировка через стабилизаторы

Иногда формулируют так: число орбит равно средней мощности стабилизатора элементов множества, но аккуратнее:

|X/G| = (1/|G|) Σx∈X |Stab(x)|.

А по лемме из предыдущего раздела это равно формуле через неподвижные точки:

(1/|G|) Σx∈X |Stab(x)| = (1/|G|) Σg∈G |Fix(g)|.

9.3. Почему деление именно на |G|

Для каждого элемента x есть фундаментальное соотношение:

|Orb(x)| · |Stab(x)| = |G|.
Доказательство формулы орбита-стабилизатор. Рассмотрим отображение φ:G→Orb(x), заданное как φ(g)=g·x. Каждый элемент орбиты получается хотя бы одним действием. Сколько действий дают один и тот же элемент орбиты? Пусть g·x = h·x. Тогда h^{-1}g·x=x, то есть h^{-1}g∈Stab(x). Отсюда g ∈ h Stab(x). Значит, прообраз каждого элемента орбиты имеет размер |Stab(x)|. Все |G| действий разбиваются на блоки одинакового размера |Stab(x)|, по одному блоку на элемент орбиты. Следовательно, |G| = |Orb(x)|·|Stab(x)|.

9.4. Доказательство Бёрнсайда через орбиты

Нужно доказать:

|X/G| = (1/|G|) Σx∈X |Stab(x)|.
Разобьём множество X на орбиты. Рассмотрим одну орбиту O. Для любого x∈O по формуле орбита-стабилизатор:
|Stab(x)| = |G| / |O|.
Внутри орбиты |O| элементов, поэтому вклад всей орбиты в сумму стабилизаторов:
Σx∈O|Stab(x)| = |O| · (|G|/|O|) = |G|.
То есть каждая орбита даёт ровно |G| в сумму. Если орбит m, то вся сумма равна m|G|. Делим на |G| и получаем m, то есть число орбит.

9.5. Доказательство через таблицу «действия × элементы»

Это наиболее полезное доказательство для задач. Мы считаем пары (g,x), где действие g фиксирует объект x.

P = { (g,x) | g∈G, x∈X, g·x=x }.

Количество таких пар по действиям:

|P| = Σg∈G|Fix(g)|.

По лемме Бёрнсайда число орбит — это среднее число объектов, фиксируемых элементом группы:

|X/G| = |P| / |G|.

Интуитивно каждую орбиту таблица считает ровно |G| раз, поэтому после деления на |G| каждая орбита даёт вклад 1.

10. Как применять лемму Бёрнсайда в задачах

Алгоритм применения:
  1. Определить множество объектов X, которые считаем без учёта симметрий.
  2. Определить группу симметрий G.
  3. Для каждого действия g∈G посчитать |Fix(g)|.
  4. Просуммировать |Fix(g)| по всем g.
  5. Разделить на |G|.

В задачах удобно выбирать, по чему итерироваться: по элементам группы или по объектам. Формулы эквивалентны:

Σg∈G|Fix(g)| = Σx∈X|Stab(x)|.

Обычно объектов намного больше, чем симметрий, поэтому чаще итерируются по действиям группы. Например, раскрасок k^n, а поворотов всего n.

Не нужно делить общее число объектов на размер группы напрямую. Это работает только если все орбиты имеют одинаковый размер. Но симметричные объекты имеют большие стабилизаторы и меньшие орбиты. Лемма Бёрнсайда автоматически учитывает это.

11. Теорема Пойа как следствие леммы Бёрнсайда

11.1. Раскраски и перестановки позиций

Пусть есть n позиций и k цветов. Множество объектов X — все раскраски позиций. Значит, без учёта симметрий объектов k^n.

Пусть группа G действует на позициях, то есть каждый элемент g∈G — перестановка позиций. Она автоматически действует на раскраски: после перестановки цвет с одной позиции переезжает на другую.

11.2. Количество неподвижных раскрасок

Если перестановка g имеет c(g) циклов, то раскраска фиксируется ею тогда и только тогда, когда все позиции внутри каждого цикла имеют один и тот же цвет.

|Fix(g)| = k^{c(g)}.

11.3. Формула Пойа для обычных k цветов

Теорема Пойа, базовая форма:
число раскрасок с точностью до G = (1/|G|) Σg∈G k^{c(g)}.
Доказательство. Это прямая подстановка в лемму Бёрнсайда:
|X/G| = (1/|G|) Σg∈G|Fix(g)|.
Для раскрасок |Fix(g)| = k^{c(g)}, потому что каждый цикл должен быть одноцветным, и для каждого цикла есть k вариантов цвета. Получаем формулу.

11.4. Более общая формула с циклами

Если нужно не просто количество раскрасок в k цветов, а учитывать, сколько раз использован каждый цвет, используют цикловой индекс. Для перестановки с m_1 циклами длины 1, m_2 циклами длины 2 и так далее записывают моном:

a_1^{m_1} a_2^{m_2} ... a_n^{m_n}.

Цикловой индекс группы:

Z_G(a_1,...,a_n) = (1/|G|) Σg∈G a_1^{m_1(g)} a_2^{m_2(g)} ... a_n^{m_n(g)}.

Для простого подсчёта в k цветов можно подставить a_1=a_2=...=k. Тогда моном превращается в k^{m_1+m_2+...}=k^{c(g)}.

12. Задача об ожерельях

12.1. Постановка

Есть ожерелье из n бусин. Каждая бусина красится в один из k цветов. Две раскраски считаются одинаковыми, если одна получается из другой поворотом.

Множество объектов X — все массивы длины n над k цветами. Группа G=C_n — циклические сдвиги на 0,1,...,n-1 позиций.

12.2. Почему циклические сдвиги — группа

Сдвиг на a и потом на b равен сдвигу на a+b mod n. Нейтральный элемент — сдвиг на 0. Обратный к сдвигу на a — сдвиг на n-a. Значит, это замкнутая группа.

12.3. Число циклов у сдвига

Рассмотрим сдвиг на t позиций. Он переводит позицию i в i+t mod n. Циклы этой перестановки имеют длину n / gcd(n,t), а число циклов равно gcd(n,t).

Доказательство. Стартуем из позиции i и многократно прибавляем t по модулю n. Вернёмся в i, когда m t ≡ 0 mod n. Минимальное положительное такое m равно n / gcd(n,t). Значит, длина каждого цикла такая. Всего позиций n, поэтому число циклов равно n / (n/gcd(n,t)) = gcd(n,t).

12.4. Формула для ожерелий с учётом только поворотов

По Бёрнсайду:

N(n,k) = (1/n) Σt=0n-1 k^{gcd(n,t)}.
Пример: n=4, k=2.
Сдвиг tgcd(4,t)Неподвижных раскрасок
042⁴ = 16
112¹ = 2
222² = 4
312¹ = 2
Итого:
(16+2+4+2)/4 = 6.
Значит, бинарных ожерелий длины 4 с точностью до поворота — 6.

12.5. Формула через делители

Часто сумму группируют по значению d=gcd(n,t). Количество t, для которых gcd(n,t)=d, равно φ(n/d). Тогда:

N(n,k) = (1/n) Σd|n φ(n/d) k^d.

Эквивалентно заменой q=n/d:

N(n,k) = (1/n) Σq|n φ(q) k^{n/q}.

13. Раскраски прямоугольника по модулю вертикального и горизонтального отражений

13.1. Постановка

Есть прямоугольная таблица размера n×m. Каждую клетку можно покрасить в один из k цветов. Две раскраски считаются одинаковыми, если одна получается из другой отражением относительно вертикальной оси, горизонтальной оси или их композицией.

Группа симметрий состоит из четырёх элементов:

ЭлементДействие
eничего не делать
Vотразить относительно вертикальной оси
Hотразить относительно горизонтальной оси
VHсделать оба отражения, то есть поворот на 180° для прямоугольной сетки

Эта группа изоморфна C_2×C_2. У неё размер 4.

13.2. Метод

Нужно посчитать для каждого из четырёх действий количество раскрасок, которые оно фиксирует. Если действие разбивает клетки на c циклов, то неподвижных раскрасок k^c.

13.3. Тождественное действие

Ничего не меняет. Каждая клетка — отдельный цикл. Всего циклов:

nm.

Значит:

|Fix(e)| = k^{nm}.

13.4. Вертикальное отражение

Вертикальное отражение меняет столбец j на столбец m+1-j. Строка остаётся той же.

В каждой строке столбцы разбиваются на пары, а если m нечётно, центральный столбец остаётся на месте. Число независимых позиций в одной строке равно ceil(m/2). Для n строк получаем:

c(V) = n · ceil(m/2).

Значит:

|Fix(V)| = k^{n·ceil(m/2)}.
Если m=5, столбцы группируются так: 
1 ↔ 5, 2 ↔ 4, 3 ↔ 3
То есть в каждой строке 3 независимых выбора цвета.

13.5. Горизонтальное отражение

Аналогично горизонтальное отражение меняет строку i на строку n+1-i, а столбец оставляет тем же.

Число циклов:

c(H) = ceil(n/2) · m.

Значит:

|Fix(H)| = k^{ceil(n/2)·m}.

13.6. Композиция вертикального и горизонтального отражений

Действие VH переводит клетку (i,j) в:

(n+1-i, m+1-j).

Это поворот на 180° вокруг центра прямоугольника. Почти все клетки образуют пары. Одиночная неподвижная клетка существует только если и n, и m нечётны, то есть есть центральная клетка.

Число циклов для такого действия равно:

c(VH) = ceil(nm/2).

Почему? Действие имеет порядок 2: каждая клетка либо фиксируется, либо образует пару. Если nm нечётно, есть одна центральная клетка и остальные пары. Если nm чётно, все клетки разбиваются на пары. В обоих случаях число циклов равно ceil(nm/2).

Значит:

|Fix(VH)| = k^{ceil(nm/2)}.

13.7. Итоговая формула

По лемме Бёрнсайда число раскрасок с точностью до этих отражений равно:

Ans = (1/4) · ( k^{nm} + k^{n·ceil(m/2)} + k^{ceil(n/2)·m} + k^{ceil(nm/2)} ).

13.8. Проверка на маленьком примере

Пусть n=1, m=2. Это две клетки в ряд. Формула даёт:

Ans = (1/4)(k^2 + k^{1·ceil(2/2)} + k^{ceil(1/2)·2} + k^{ceil(2/2)}) = (1/4)(k^2 + k + k^2 + k) = (k^2+k)/2.

Это верно: раскраска двух клеток с точностью до отражения — это неупорядоченная пара цветов. Таких пар с повторениями: k(k+1)/2.

14. Шпаргалка

Основные определения

ТерминСмысл
ГруппаМножество действий с композицией, нейтральным элементом и обратными.
Действие группыПравило g·x, переводящее объект в объект.
ОрбитаВсе объекты, достижимые из данного действиями группы.
СтабилизаторДействия, оставляющие объект неподвижным.
Неподвижные точкиОбъекты, которые фиксируются данным действием.

Формулы

Orb(x) = {g·x | g∈G}
Stab(x) = {g∈G | g·x=x}
Fix(g) = {x∈X | g·x=x}
Σx∈X|Stab(x)| = Σg∈G|Fix(g)|
|Orb(x)|·|Stab(x)| = |G|
|X/G| = (1/|G|)Σg∈G|Fix(g)|
для раскрасок: |Fix(g)| = k^{число циклов g}
Пойа: |X/G| = (1/|G|)Σg∈Gk^{c(g)}

Что чаще всего спрашивают

  • Почему нельзя просто делить на размер группы? Потому что орбиты могут иметь разный размер.
  • Что считает Fix(g)? Объекты, не меняющиеся после конкретной симметрии.
  • Почему в Пойа появляется k^{c(g)}? Потому что каждый цикл перестановки должен быть одноцветным.
  • Почему у сдвига на t циклов gcd(n,t)? Потому что длина цикла равна n/gcd(n,t).
  • Что делать в задачах? Выписать группу симметрий, посчитать число циклов/фиксированных объектов для каждого действия, усреднить.

Мини-шаблон решения задачи

1. X = множество всех объектов без учета симметрий.
2. G = группа допустимых симметрий.
3. Для каждого g из G:
      найти, какие условия накладывает g·x = x;
      посчитать |Fix(g)|.
4. Ответ = сумма |Fix(g)| / |G|.
