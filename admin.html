// github pull request flow

Добавить конспект

Заполните карточку, загрузите HTML-файл и отправьте заявку. Сервер создаст Pull Request в GitHub: конспект появится на сайте только после проверки и merge.

Предпросмотр

Что произойдет после отправки
  • создастся новая ветка в GitHub;
  • HTML-файл попадет в папку дисциплины;
  • обновится `data/lectures.json`;
  • откроется Pull Request для ревью.
