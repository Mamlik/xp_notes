# Конспекты МИФИ · Экстремальное программирование

Статический сайт с конспектами по компьютерным дисциплинам программы
«Экстремальное программирование» в МИФИ.

## Структура

```text
index.html                 Главная страница библиотеки
admin.html                 Заглушка будущей админки
assets/
  app.js                   Рендер карточек и поиск
  catalog.js               Фолбэк каталога для открытия сайта как файла
  styles.css               Общие стили сайта
  mephi-logo-cropped.jpg   Логотип в шапке
data/
  lectures.json            Каталог дисциплин и конспектов
lectures/
  acos/                    Конспекты по АКОС
  algorithms/              Будущий раздел по алгоритмам
  distributed/             Будущий раздел по распределенным системам
```

## Как добавить конспект вручную

1. Положить HTML-файл в нужную папку внутри `lectures/`.
2. Добавить запись в `data/lectures.json`.
3. Убедиться, что `href` указывает на новый HTML-файл.
4. Обновить `assets/catalog.js`, если сайт должен корректно открываться как
   локальный файл без веб-сервера.
5. Открыть сайт через локальный или боевой веб-сервер.

Главная страница строит карточки из `data/lectures.json`. Если браузер блокирует
локальный `fetch`, используется фолбэк из `assets/catalog.js`.

## Идея будущей админки

Админка создает Pull Request в GitHub: добавляет HTML-файл в `lectures/` и
обновляет `data/lectures.json`. Публикация проходит через ревью, а история
изменений остается в Git.

## Запуск на VPS

Нужен Node.js 20+.

```bash
cp .env.example .env
npm start
```

Переменные окружения:

```text
PORT=8080
ADMIN_PASSWORD=пароль-для-формы-админки
GITHUB_OWNER=владелец-репозитория
GITHUB_REPO=имя-репозитория
GITHUB_TOKEN=github-token
GITHUB_BASE_BRANCH=main
```

Для `GITHUB_TOKEN` лучше использовать fine-grained token с доступом только к
этому репозиторию. Нужные права: `Contents: Read and write` и
`Pull requests: Read and write`.

На боевом VPS стоит поставить Nginx перед Node-приложением и включить HTTPS.
Пароль администратора передается только на сервер, но его все равно нельзя
гонять по открытому HTTP.
